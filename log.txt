Python script analysis_start downloads data prepared in a dedicated folder
according to a pre-defined scenario. Parameters are specified in a
configuration file. Scenario has to be selected by an argument.

optional arguments:
  -h, --help            show this help message and exit
  -s SCENARIO, --scenario SCENARIO
                        Sets an analysis to a given scenario. The scenario has
                        to be one from an existing ones.
  -r RADAR, --radar RADAR
                        Selects a radar(s) to process, one or both from L, R.
                        Write L to process left radar, R to process right one
                        or B to process both of them
  -b BEAM, --beam BEAM  Selects a beam(s) to process, one or more from 0,1,2,3
  -d DATASET, --dataset DATASET
                        Selects a dataset to process, the new one or the old
                        one
  -l, --list            Prints a list of available scenarios
  -o OUTPUT, --output OUTPUT
                        Sets path to the folder where output files will be
                        stored.
